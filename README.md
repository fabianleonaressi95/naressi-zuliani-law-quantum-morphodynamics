# Unified Hamiltonian Simulation

Repository per la validazione computazionale delle interazioni tra la metrica ottica della sfera fotonica (spaziotempo curvo) e le risonanze fononiche in reticoli aperiodici (quasicristalli).

## Framework Teorico
Il modello implementa le costrizioni geometriche della **Legge Quantistica di Campo di Naressi**, applicando lo sfasamento costante della Sincronizzazione Aurea:
$$\theta_\phi = 2\pi(1 - 1/\phi)$$

Il profilo di accoppiamento dinamico sopprime le divergenze ultraviolette tramite il damping deterministico:
$$J_n = \phi^{-n} e^{-n/\phi^2}$$

## Requisiti e Uso
Installa le dipendenze con `pip install -r requirements.txt` ed esegui `python simulation.py`.
